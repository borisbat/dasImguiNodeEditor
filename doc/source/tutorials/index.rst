###############################
  dasImguiNodeEditor tutorials
###############################

Step-by-step walkthroughs of the node-editor surface. Each tutorial pairs an
annotated source file under ``examples/tutorial/`` with prose narrating the
construction, plus a short screen recording produced by a Playwright-style
harness driver. Interaction tutorials are additionally backed by a headless
regression that drives the same gesture (e.g. ``test_connect_drag`` for the drag
below).
